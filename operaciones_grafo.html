MÓDULO OPERACIONES GRAFO¶

Módulo contenedor de operaciones de grafos.

El coste de cada operación OE se encuentra como comentario en cada línea del código fuente.

operaciones_grafo.ejecutar_kruskal(lista_grafos)¶

Ejecuta kruskal de una lista de grafos

operaciones_grafo.ejecutar_nodo_central(lista_grafos)¶

Ejecuta el algoritmo de nodo central de una lista de grafos

operaciones_grafo.ejecutar_prim(lista_grafos)¶

Ejecuta prim de una lista de grafos

operaciones_grafo.generar_fichero(lista_grafos)¶

Genera un fichero conteniendo la información solicitada

operaciones_grafo.imprimir_imagen(lista_grafos, extra='')¶

Imprime una imagen a partir de una lista de grafos dada

operaciones_grafo.imprimir_nodos_centrales(lista_grafos)¶

Genera un fichero conteniendo la información solicitada

operaciones_grafo.leer_grafo(ficheros)¶

Lee los ficheros introducidos y devuelve una lista de grafos

operaciones_grafo.sobreponer_grafos(lista_grafos, extra='')¶

Imprime varios grafos en el mismo lienzo. El primer argumento determinará el nombre y tamaño del lienzo

operaciones_grafo.unir_grafos(lista_grafos)¶

Une los grafos de una lista de grafos de forma rápida Complejidad: T(n) = 2 + g*(2 + 1 + 2 + 1 + (1+c) + m*(1+d) + n*(1 + (1+e))) + 1 = 8g+2m*g+3n*g+3
