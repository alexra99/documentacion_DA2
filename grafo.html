get_adjacent(nodo)¶

Devuelve los nodos adyacentes al dado Complejidad: T(n) = 1 + n*(2 + m*(4 + 1)) + 1 = 5n*m+2n+2

get_adjacent_and_edges(nodo, padre)¶

Similar a get_adjacent. Devuelve los nodos y arcos adjacentes a un nodo dado Complejidad: T(n) = 1 + n*(2 + m*(6 + 2)) + 1 = 8n*m+2n+2

get_edge(nodo1, nodo2)¶

Si existe, devuelve el arco entre dos nodos, si no, devuelve 0 Complejidad: T(n) = n*(5 + 1) + 1 = 6n+1

get_internal_nodes()¶

Obtiene los nodos internos Complejidad: T(n) = 1 + m*(1 + 1) + 1 = 2m+2

get_matriz(node)¶

kruskal()¶

matriz_adyacencia(internal)¶

Genera la matriz de adyacencia del grafo Complejidad: T(n) = 1 + m*(2+c) + 1 = 8n*m^2+2n*m+6

obtener_distancia(nodo, padre)¶

Obtiene la distancia total de un nodo a todas las hojas Complejidad: T(n) = 1 + 2 + 1 + m*(2 + 8 + n*(m+n*log(n))) + 1 = n*m^2+m*n^2*log(n)+10m+5

pop_minor_node()¶

Devuelve y elimina (pop) el nodo con menor distancia de la lista de nodos Complejidad: T(n) = 2 + m*(1 + 1) + 1 + 1 = 2m+4

prim()¶

Ejecuta el algoritmo de prim en el grafo T(n) = 1 + 1 + m*((2+c) + 1 + n*((1+d) + 2 + (3+e) + 1 + 1 + 1 +2)) + 1 + 1 + m*(1 + (2+f)) + 1 = 10n^2*m^2+8n^2*m+2m^2+17n*m+11m+5

set_leaves()¶

Setea las hojas de un grafo (nodos con un solo arco conectado) Complejidad: T(n) = m*(1 + 1 + n*(2 + 1 + 1)) = 4n*m+2m

sort_edges()¶

Ordena los arcos del grafo de menor a mayor Complejidad: T(n) = 1 + n*(2 + n*2) + 1 + 1 = 2n^2+2n+3
